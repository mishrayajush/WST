This repository contains multiple JavaScript and HTML practice programs created while learning core Web Technology concepts.
It includes small standalone scripts, mini examples, and basic Node.js practice files.

The goal of this repo is learning and experimenting, not a single production app.

 Project Structure

The project contains different types of files:

 JavaScript Practice Files

These files demonstrate core JavaScript concepts such as:

Functions
Objects & Arrays
Console operations
Simple logic building
Mini problem solving programs

Examples:

calculator.js → Basic calculator logic
console.js / console2.js → Console-based programs
function add(a, b).js → Function usage example
const students.js → Array of objects example
const cart.js → Shopping cart object example
const travel.js → Travel object data example
const airline.js → Airline object example

Many files have duplicates with (1) — these are backup copies created during editing.

 HTML Pages

Basic frontend pages used for practice.

homepage.html → Simple homepage layout
Q1.html → HTML assignment/practice page

Concepts covered:

Basic HTML structure
Forms and input elements
Page layout
 Node.js / Express Practice

This repo also includes a simple backend practice file:

const express = require('express').js

This demonstrates:

Setting up a basic Express server
Understanding backend basics
 Mini Logic Programs

Small scenario-based JavaScript programs:

hospitaladmission.js
onlineelectricity.js
read.js / read-2.js

These simulate real-world use cases using JS logic.

 Purpose of the Repository

This repository was created to practice and understand:

HTML fundamentals
JavaScript basics
Objects & arrays
Functions and logic building
Intro to Node.js & Express
Problem solving using JS

It acts as a learning sandbox for Web Technology subjects.
